# 💡 解读格式模板（v17 最终版）

## 核心规则

每条新闻的解读必须用以下紧凑 4 行格式，每行以 emoji 标签开头：

```
🔴 [新闻标题](URL)

  📍 来源#排名 · 热度

  📌 事件：发生了什么/新闻内容是什么（2-3句，给出完整背景细节）
  🌊 影响：对谁有影响、什么影响、为什么重要（2-3句，说透影响面）
  👀 后续：下一步看什么、关注什么时间节点（2-3句，给出具体跟踪方向）
  💡 建议：可操作的行动建议（2-3句，关注什么、注意什么、做什么准备）
  ───────────────────────────────────────
```

## 格式要点

1. **标签用 emoji 前缀**：`📌 事件` `🌊 影响` `👀 后续` `💡 建议`，不用 `` ` `` 反引号
2. **行间无空行** — 4 行直接换行，不要额外空行
3. **每行 2-3 句** — 确保内容充足不敷衍
4. **标题和来源分两行** — 标题行放链接，来源行用 `📍`
5. **卡片以 `───` 分隔线结尾** — 每个条目自成卡片
6. **条目之间有空行** — 不同卡片之间空一行

## 示例

```
🔴 [巴西把自己踢晕了](https://www.toutiao.com/trending/xxx)

  📍 头条#1 · 2074万

  📌 事件：世界杯1/16决赛最大冷门——日本1-0淘汰五冠巴西。巴西全场零射正、球员被曝中场内讧，日本凭借极致防守反击创造历史
  🌊 影响：全平台霸榜的头号热点。巴西出局引发连锁反应：内马尔可能告别国家队、巴西足协面临换帅、夺冠赔率剧变
  👀 后续：看日本16强对手（对阵决定能走多远）、巴西足协是否宣布换帅、内马尔国家队去留声明
  💡 建议：关注日本队战术复盘视频，巴西球员转会窗口可能提前开启，世界杯赔率需重算
  ───────────────────────────────────────
```

## 国内版（不带来源行）

```
🔴 [中国对日本再出重拳](https://www.toutiao.com/trending/xxx)

  📌 事件：中国代表在联合国大会公开点名日本二战罪行，措辞严厉近年罕见
  🌊 影响：中日外交从双边升级到多边舞台，信号意义强烈——预示中国对日政策可能转向主动施压
  👀 后续：紧盯日本外务省回应、中国是否推动联合国涉日决议
  💡 建议：中日关系敏感期，跨境贸易和投资决策需审慎评估政治风险
  ───────────────────────────────────────
```

## 常见错误

❌ **解读只有 1 行** — "该事件正在获得关注" → 不合格
❌ **解读行间有空行** — 4 行之间不要有空行
❌ **来源用"来源："** — 必须用 `📍` 前缀
❌ **标题不带链接** — 所有新闻标题必须是可点击的 Markdown 链接
❌ **字数太少** — 每条 1 句话带过 → 每行至少 2 句
