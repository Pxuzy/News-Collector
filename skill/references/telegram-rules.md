# Telegram 渠道规则

## 最高优先级

Telegram 渠道优先级高于通用 Markdown 文档规则。只要目标是 Telegram，就按本文件执行。

## 禁止项

- 不要把整篇报告放进代码块。
- 不要输出 ```md 或 ``` 代码围栏。
- 不要使用 Markdown 表格。
- 不要使用 `---` 作为分隔线；Telegram 上间隔过大，直接用栏目标题分隔。
- 不要依赖 `# / ## / ###` 标题渲染；Telegram 里更推荐 `🔥 **栏目标题**`。
- 新闻列表优先用 `•`，不要用大量 `-`，避免看起来像源码清单。
- 不要只输出纯标题。
- 不要省略链接、平台排名、热度。
- 不要编造缺失数据。

## 链接规则

- 首选：`[标题](URL)`。
- 不要在 Markdown 链接后面重复裸 URL，例如禁止 `[标题](URL) (URL)`。
- **链接 URL 必须使用百分号编码**，禁止在 URL 中放入未编码的中文、引号、括号等特殊字符。
  - ✅ 正确：`[标题](https://www.baidu.com/s?wd=%E4%B8%AD%E5%9B%BD)`
  - ❌ 错误：`[标题](https://www.baidu.com/s?wd=中国)` — Telegram 渲染会完全失败，链接不生成、括号不匹配、后续 Markdown 全部被破坏
- 如果 Telegram 仍然渲染失败，降级为：
  - `标题`
  - `🔗 URL`
- 如果没有原始链接，使用可解释的兜底链接，例如搜索链接，并在数据缺口说明。
- 如果完全没有链接，写 `链接未获取`，并把该条计入数据缺口。

## 热榜字段

热榜/趋势榜每条都要包含：

- 平台名：抖音、微博、知乎、百度、GitHub、HN 等。
- 平台排名：例如 `抖音#1`、`微博#6`、`HN#3`。
- 热度指标：例如 `热度 832万`、`Stars 18.4k`、`420 points / 88 comments`。
- 内容类型或气质：社会、娱乐、突发、AI、开源、论文、情绪共鸣、争议等。
- **每条社媒热榜条目（抖音/微博/知乎/百度）必须有 💡 解读**，说明为什么火、传播因素、后续看点。
- 社媒热榜的解读不能只写“很火”，至少说明：传播因素、影响范围、后续观察点三者之一。
- GitHub 项目不能只写仓库名和 Stars，必须说明“项目是干嘛的、适合谁、为什么热、后续看什么”。

缺失时写：

- `排名未获取`
- `热度未获取`
- `类型未判断`

热榜条目不可只给标题不给分析。没有解读的热榜就是标题搬运。

## 推荐单条格式

• 🔴 [标题](URL)
  来源：平台#排名 · 热度 · 时间 · 类型
  📌 事件：一句话说明发生了什么。
  🧭 背景：复杂事件才写，说明为什么现在重要。
  🌊 影响：影响哪些人、行业、公司、市场或技术路线。
  ⚠️ 风险：复杂或高风险事件才写，说明不确定性。
  👀 后续：下一步看什么。
  💡 为什么重要：一句话解释价值。
  ✅ 可信度：高/中/低；单源必须写“单源待核实”。

## AI 解读深度

- 普通热榜：只写 `💡 解读`，一两句话即可。
- 普通新闻：写 `📌 事件 / 🌊 影响 / 👀 后续 / 💡 为什么重要 / ✅ 可信度`。
- 复杂新闻：额外写 `🧭 背景` 和 `⚠️ 风险`。
- 重大新闻：必须写清影响对象、二级影响、后续观察指标和不确定性。
- 技术/GitHub/HN/论文：不能只写“很火”，要解释解决什么问题、适合谁、成熟度和跟踪价值。

## 分段规则

- 单条 Telegram 消息建议 3500-4000 字符以内。
- 简报因太长导致模型"后面忘了规则"。**必须逐个板块分段生成**，每个板块 4-6 条：
  1. 🧭 今日主线（3-6条）
  2. 🏮 国内热点 - 🌞 社会·民生（4-6条）
  3. 🏮 国内热点 - 🔬 科技·数码（4-6条）
  4. 🏮 国内热点 - 🎬 娱乐·综艺 + ⚽ 体育·赛事（4-6条）
  5. 🌏 国外热点 - 🌍 国际·要闻 + 💰 财经·商业（4-6条）
  6. 🐙 GitHub 趋势（日榜前10 + 周榜去重新增）
  7. 📜 论文·学术 + 🤖 AI·前沿 + 收尾栏目
  8. 合并 → validate_briefing.py → 通过后发送文件

## 每项必填

- **每个条目必须有热度值和平台排名**（如 "百度#2 · 热度781万"），不能只写标题。
- **国内热点每条必须有 💡 解读**，不能只列标题和来源。
- **主线/国际/财经条目也必须写💡**：避免validator误判。

## 语气

- 中文为主。
- 短句、强信息密度。
- 多用 emoji 做扫描锚点，但不要让 emoji 替代事实字段。
- 先给主线，再给条目。
